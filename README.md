# 🎓 Student Information Manager CLI Project

A simple Command Line Interface (CLI) application built with **Dart** for managing student information. This project demonstrates the use of Object-Oriented Programming (OOP) concepts and basic data management through an interactive terminal-based application.

---

## 🚀 Features

- Add new student records
- View all students
- Search students by name
- Update student information
- Delete student records
- Simple menu-driven interface
- Beginner-friendly implementation

---

## 🛠️ Built With

- Dart
- Command Line Interface (CLI)

## ▶️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/sharifahmedd/student-information-manager-cli-project.git
```

### 2. Navigate to the project directory

```bash
cd student-information-manager-cli-project
```

### 3. Run the application

```bash
dart run assignment.dart
```
